# Col·laborar amb REC Time Monitor

Gràcies per voler contribuir al projecte.

## Com proposar canvis

1. Obre una issue explicant el problema o la millora proposada.
2. Fes un fork del repositori i crea una branca específica per al canvi.
3. Mantén els canvis centrats en una sola funció o correcció.
4. Comprova que el projecte compila a macOS i/o Windows segons l'àrea afectada.
5. Envia una pull request explicant què canvia i com ho has provat.

Per als errors, indica la versió d'OBS, el sistema operatiu, els passos per
reproduir-los i, si és possible, adjunta el registre d'OBS.

Les contribucions acceptades es publiquen sota la llicència GNU GPL v2 del
projecte.
